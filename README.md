# SmartDiary

## Overview
It is a smart diary 
It's a diary, but it's a useful program that allows you to manage your schedule and add monthly revenue and expenses to your diary.

### Using Library
- JFxtras

- Apache Common Codec

- JFoenix

### Development Environments (Windows)
- NetBeans IDE 8.2

- Windows 10

### Development Environments (Linux)
- IntelliJ IDEA

- Ubuntu Linux 16.04
